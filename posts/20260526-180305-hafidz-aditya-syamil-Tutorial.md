Selamat datang di WASPADA Blogspot, tempat mengumpulkan dan membaca tugas bahasa secara rapi dalam satu halaman. Untuk mulai menggunakan website ini, scroll dari halaman pembuka ke bagian utama, lalu pilih tindakan sesuai kebutuhan: tekan tombol Create jika ingin menulis posting langsung di website, pilih nama kamu terlebih dahulu, isi judul dan tulisan dengan jelas, lalu simpan agar posting muncul di daftar; tekan tombol Upload jika tugas kamu berbentuk PDF, pilih nama kamu, lalu seret file PDF ke area upload atau klik area tersebut untuk memilih file dari perangkat, pastikan file yang dipilih benar dan ukurannya tidak melebihi batas yang ditentukan; setelah proses upload atau penyimpanan selesai, tugas akan muncul sebagai kartu di halaman utama berisi judul, nama penulis, jenis file, dan waktu pembaruan. Untuk membaca tugas, cukup klik salah satu kartu yang tersedia. Jika ingin melihat tugas berdasarkan siswa tertentu, gunakan tombol Student, pilih nama siswa, dan website akan menampilkan hanya tugas dari nama tersebut; filter bisa dihapus kembali melalui tanda silang pada chip filter. Untuk guru, tombol Teacher View dapat digunakan agar posting dikelompokkan berdasarkan nama siswa sehingga lebih mudah diperiksa satu per satu. Pastikan selalu memilih nama yang benar sebelum membuat atau mengunggah tugas, gunakan judul yang mudah dikenali, dan tunggu sampai proses selesai sebelum menutup popup agar file atau tulisan tersimpan dengan baik.
